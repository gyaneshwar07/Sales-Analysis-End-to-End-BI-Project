üí° Project Overview
This project involved developing a comprehensive Star Schema Data Model and conducting Sales Data Analysis for ElectroHub, a retail company dealing in categories like Electronics, Footwear, Clothing, and Home Appliances. The core objective was to transform raw data into a reliable and interactive reporting solution, enabling quick, data-driven decisions.

üõ†Ô∏è Technical Deep Dive
Modeling Concept: Implemented a robust Star Schema architecture (Fact Table: Sales; Dimensions: Product, Customer, Promotion).

Data Prep: Focused on detailed Data Profiling and Data Transformation.

Database Fundamentals: Explicitly defined Primary & Foreign Keys and Cardinality (1-to-Many relationships) to ensure data accuracy.

Calculations: Utilized DAX for advanced measures and comparative analysis .

üìä Key Business Requirements Fulfilled
The final reports provided insights on the following eight critical areas:

Product Performance: Top/Bottom 5 products by Sales, Profit, and Quantity Sold.

Profitability: Visualizing the relationship between Sales & Profit.

Period Comparison: Comparing Sales/Profit/Quantity between any two user-selected periods.

Discount Analysis: Calculating the Average Discount offered per discount category.

Orders: Total number of orders placed.

Order Details: Detailed order-level view of Sales, Profit, Discount, and Net Sales, with dynamic visual filters (Product, Date, Customer ID, Promotion).

Geographic Sales: Sales performance mapped across different cities.
